## SIDELOAD
Releases にある mayq-0_8_0.dsk ファイルをダウンロード、[Apple\]\[js](https://www.scullinsteel.com/apple2/) などの AppleII 環境で動かしてください。

## 操作
- `W`, `S`, `A`, `D`<br>上下左右に移動、停止
- `I`, `K`, `J`, `L`<br>上下左右に移動、停止
- `1`, `2`, `3`<br>ゲーム速度の変更
- `ESC`<br>ゲームの保存
